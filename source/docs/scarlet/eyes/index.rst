Eyes in the world tree
=========================================================

.. image:: /_static/images/tree-eyes.png
  :alt: World tree

A hijack unseen is a hijack untrusted. A router knows what it announced, but it can never tell from its own branch
whether the rest of the network believed the update or discarded it as nonsense.

Seeing past that isolated perch requires borrowing public eyes: passive collectors, looking glasses, and monitors.
They require zero target contact, but the view cuts both ways: the same public records used to verify propagation
are the ones defenders watch to catch the shift.

How to borrow external sight, read the historical record, alert on changes, and build a private listener to avoid
leaving a query trail in someone else's logs.
