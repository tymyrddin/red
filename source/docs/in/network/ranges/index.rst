Ranges
======

Three external lab environments, collected here by administrative fact rather than pedagogical similarity.

TryHackMe offers guided learning paths with scaffolded rooms and worked progressions. Root-me presents unsolved
challenges where the approach is left to the reader. RIPE NCC provides exercises built around actual internet
infrastructure, routing tools, and real-world measurement systems. Each is worth using; none is a substitute for
the others. The attack surfaces these labs exercise are included in Rootways and Patches; the operational moves
they develop can be found in Tradecraft.
