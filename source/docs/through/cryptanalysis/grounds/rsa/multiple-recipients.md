# RSA Multiple recipients

[RootMe Challenge: Chinese!](https://www.root-me.org/en/Challenges/Cryptanalysis/RSA-Multiple-recipients): A message sent to 3 different people has been intercepted. We have the public keys of those people. Find the message to validate the challenge.
